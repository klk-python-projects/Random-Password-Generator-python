# Random Password Generator 🔐

A simple Python-based Random Password Generator that creates secure and random passwords based on the user's desired password length.

## Features

- Generates random passwords
- User-defined password length
- Uses uppercase and lowercase letters
- Includes numbers
- Includes special characters
- Input validation
- Exception handling
- Option to generate another password

## Technologies Used

- Python
- Random Module
- String Module

## How It Works

1. The user enters the desired password length.
2. The program combines letters, numbers, and special characters.
3. A random password is generated.
4. The user can choose to generate another password.
